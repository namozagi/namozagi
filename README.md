## 🚀 كيفية الاستخدام

### الطريقة 1: تشغيل مباشر (الأسهل)
1. افتح ملف `index.html` مباشرة في متصفحك (Chrome, Firefox, Safari, Edge)
2. الموقع سيعمل بدون الحاجة لأي خادم ويب

### الطريقة 2: استخدام خادم محلي (خيار أفضل)

**على Windows:**
```bash
# استخدم Python (مثبت على معظم الأجهزة)
python -m http.server 8000
# أو
python -m SimpleHTTPServer 8000

# ثم افتح المتصفح على: http://localhost:8000
```

**على Mac/Linux:**
```bash
# استخدم Python
python3 -m http.server 8000

# أو استخدم Node.js (إذا كان مثبتاً)
npx http-server

# ثم افتح المتصفح على: http://localhost:8000
```
